# 教育



## 心得与分享
### 2024

[202410232349] 作为一名终身学习的信仰者，真正涉足教育“算是”三年有余了，但总体的感受是“痛苦”的，也有些“煎熬”；之前经常提到，让学生去学习某些东西或者做某件事（特别是学生不去做或者没有主观意愿的情形下），让我的内心受到一种伦理上的困扰；特别是为人父之后，对教育也有了更多的尊重和思考；我一直认为有了一定基础（这个“基础”后续再深入思考）之后，学习是一件个人、自主的事情；如何达到让自己满意、舒服的教学状态，我现在心理没有底，甚至已经做好了退却的准备；但无论如何，每个人都需要去做些自己感觉有价值的事，教师也是如此，如果只是单纯传授自己一辈子也用不到的知识和技能，这样的教学大概没有什么意义；对于教师，做好自己的事情，做好榜样和陪伴，也许这就是最好的教学状态了，教师与学生可以相互成就，共同丰富彼此的人生。
[202410250708] 当教师是一件很挫败的事情。不过也遇到很意思的现象，在QLNU，在我看来根本不能算合格的学生，职业规划中最多的选择是当教师，这是一件值得思考的事情。真正参与教育的三年以来，我“浪费”了太多时间在“形式”上，结论就是这些时间是真的浪费了，特别是浪费了做“真事”的时间；有一些事情不断反复的跟学生强调，还是没有响应或行动，有时以为是没有通知到位，也不断反复跟学习委员（班委）嘱咐，还是没有变化，这只能说明没有必要这样做下去了。从今天起，做一个让自己幸福的人。
[202412121733] 学校教育存在诸多问题，有很多“形式化”的东西，但在社会或者工作等情形中也类似，作为在校学生，首先要保证正常的教学（学习）秩序，“认真”完成基本的东西，才有更多的时间、精力和可能去突破和完善自己。

### 2023

[202303062323] 所谓的名校，所谓的知识分子，所谓的有钱人，所谓的上层社会，所谓的看上去光鲜的一类机构或者一类人，都是一群装逼的人。只是有的人可能"开窍”比较早罢了，只是有的人幸运一点而已多赚了点儿纸币。所处的环境，所带的标签，都不可能去定义一个人，更没必要自我贬低，只要不断地努力、思考、实践，不卑不亢地前行，这样的人生就是有价值的；人与人的差别是很微小的，洗尽铅华，每个人一生都只是做了一点点，可能属于自己真正的东西。如果有一天还在后悔，那只是因为在不断重复自己的问题，如果说有遗憾的话，这就是所可惜的地方了。
[202305300931] 教育的目的是用一束光去点亮一个人，虽然我也打过很多次火。成长不一定发生在学校，但学校却是一个可以让人快速成长的地方。用一束微光，持续照亮一群老6。
