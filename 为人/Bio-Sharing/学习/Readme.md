## 表现
#奖学金
进入大学阶段后，每个学校都有各类奖学金，可谓琳琅满目，但大部分奖学金与中学阶段不同，中学阶段大部分奖学金只看成绩，如果能拿到前几名，也不用评比、应该可以拿到不错的奖学金。
大学阶段课程的成绩，总体来说很难去横向评价不同人的真实表现；相应地，大学阶段的各类奖学金也会衡量成绩以外的其它表现，所以评比过程会有不少”主观“因素，一定程度上这也是合理的。
大学阶段的奖学金，如果有机会、个人也有意愿，应该要认真准备，积极争取，但要有一颗平常心。如果能够拿到奖学金，偷偷乐一下即可，不要太认真，它跟一个人的真正能力关系甚微；如果拿不到，继续去学习、追求、争取提升自己就可以了。
