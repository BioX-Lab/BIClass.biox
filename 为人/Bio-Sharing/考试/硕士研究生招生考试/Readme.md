# 硕士研究生招生考试
#考研




## 报名

考研与高考的区别是考前两个月左右提前报名、选择目标院校，这种方式对于犹豫不决或者对自我认识不充分的学生有些困难。
在报名这个时间节点，大部分同学已经围绕“梦中情校”做了一些准备工作，一般来说不用太犹豫，只要自己感觉良好，可以继续坚持目标；如果冷静思考后，可以发现自己的相关问题，鼓励大家适当降低“情校”的标准，以稳为主，绝大部分院校或者的导师的差异不大、并且有一定的质量保证，研究生阶段能否提升自我关键看自己的所做所为，如果真的有“天赋”或者想继续深造，硕士阶段可以做为一个跳板；如果还是瞻前顾后，鼓励继续坚持原目标，即使完成了报名，仍然有两个月左右的时间去准备，理论上时间是充分的。

BI2021@QLNU是BIers@QLNU中第一届参加考研的，期待每个人都认真准备，有个理想的结果。
