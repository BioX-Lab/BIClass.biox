## 心得与分享

### 2024

[202412211846] 这两天BI2021@QLNU正在参加研究生入学考试，这是一个重要的考试、但又真的不太重要（后续再说），希望每一个BI2021er@QLNU可以保持正常心态，把正常的自己表现出来（即可）。关于SIG和其它课程项目，之前一般会在项目中添加comment，但意义不大，还经常让自己很生气，我个人也在不断调整，后续会系统迁移到BIClass，争取平静地把反馈传达，哪怕同样的问题、一次又一次；BI@QLNU无论做多久，至少要有些收获，不留太多遗憾。
[202412242050] 最近BI2021@QLNU考研结束了，但从我所带的几个毕业论文、SIG-5等进展来看，可能BI2021@QLNU还是有些小思绪。考试已经结束了，就不要想太多了，继续向前走，利用好每一天，如果每天都可以清楚自己要（学）做什么，肯定比研究生学历重要的多。
每个人都有缺点，只能依靠我们自己去感悟、改变，虽然很难，但只要坚持总结、反思，肯定会有变化的，未来会变得更好一些。
对于BI2021@QLNU的考研过程等，我还是很关注的，但好像大家都不太需要我的协助，包括志愿或者备考等，虽然我内心有些失落，但能够接受；今年的天气总体还是不错的；我能够预感到大家的成绩不会很理想，所以考试一结束，我就在BIClass提到了、先找一个合适的工作，继续把剩余的学业和时间利用好。
考研的过程应该是很痛苦的，可能我的记忆已经模糊了，但我可以感受到现在学习“条件”应该会好一些；备考过程会让人产生各种感受和领悟，但总体来说，不要把这段过程看的太重要，它不会留下太多东西，对于一个人的提升没有实质的帮助，无论考上与否，对一个人真正的能力和认知不会有本质的变化，我在BIClass commit message中也提到了，这是一段不正常的经历或状态；把接下来的每一天利用好更重要些。
对于考研结果，不要忧虑太多，无论第一志愿是否可行，继续好好学习、实践，即使第一志愿不成功，有合适的调剂选项，肯定是应该要走的；硕士在哪不是很重要、差别不大，先去读，再继续提升。
对于工作肯定要找的，有个保底的工作，然后再继续前行，如果工作不错，未必要去读研究生。
现在已经过了靠学历可以找个好工作的时代（以后我应该会继续分享），重要的是个人的方向选择、能力（包括学习能力、做事能力等）等，生信还是有很多潜力方向的，作为一生或一段人生的事业应该是个不错的选择。
继续加油吧，还是“学习、做事、做人”，只是我们不断践行，是不会差的。
最近我一直在思考与BI@QLNU的相处之道，以前各种原因确实follow得不够全面和深入，现在想来有些后悔，也会把过去的点滴和遗憾不断弥补上；从今天开始我也会坚持每天集中在22:00-22:40之间维护BIClass、认真感受BIers@QLNU；前几天时不时去面对一群装睡的BIers@QLNU而去更新BIClass等、是真的会死人的；我还有很多事情需要去实现、去拓展，需要专注，之前在与BI@QLNU相处过程中，确实浪费了太多无谓的时间和精力等；我从不怀疑BI在QLNU是"最好"的，因为我想打造的BI是一个让人成长的专业、一个重视学习和做事能力的专业，老师们在践行着同样的东西、也会不断把感悟分享给BIers@QLNU，我在为此努力、实践和完善着。
[202412311058] 静水流深，新岁序开；Happy New Year!
### 2021

[202112270201] 很高兴昨天晚上全程参加了BI2021的元旦晚会。大家的表现真的很棒，跳舞、乐器、唱歌、相声、小品，还有更重要的真诚与情义。之前有点犹豫是否参加，但我知道、如果不参加，我会后悔的。我内心是很喜欢这种活动的，特别是我的BI小可爱们组织的，很高兴这次记忆也有我的参与。可能我们大学期间组织的活动会很少，这一次也算是一个比较珍贵的机会, 接下来我们都将全力以赴。这里做个约定,等明年疫情结束了，大概也是这个时间段，我请大家去海底捞，在与BI2021相识的这100多天内，我主要的精力和期待，都是在培养大家好的习惯、可以学会管理、学会主动学习等；虽然我会督促大家去准备英语等级考试，计算机等级考试，期末考试等，但我始终坚信，衡量大家价值和成功的地方不在于考试的成绩, 而在于是否发现了自己的潜力和兴趣，并积极主动的为之坚持与奋斗；接下来的时间我也会朝着这个目标不断去努力，为大家创造各种机会与条件，也希望能够看到大家都不断进步；人生的旅途中每一天都算数，希望大家珍惜好当下的每一刻，健康、快乐地奔向自己更加美好的未来，不只是大学四年，也希望在接下来的每一个阶段，都能够与大家分享彼此的体验，分享彼此的酸甜苦辣。
