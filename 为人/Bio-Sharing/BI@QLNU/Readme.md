### 2021

[202112270201] 很高兴昨天晚上全程参加了BI2021的元旦晚会。大家的表现真的很棒，跳舞、乐器、唱歌、相声、小品，还有更重要的真诚与情义。之前有点犹豫是否参加，但我知道、如果不参加，我会后悔的。我内心是很喜欢这种活动的，特别是我的BI小可爱们组织的，很高兴这次记忆也有我的参与。可能我们大学期间组织的活动会很少，这一次也算是一个比较珍贵的机会, 接下来我们都将全力以赴。这里做个约定,等明年疫情结束了，大概也是这个时间段，我请大家去海底捞，在与BI2021相识的这100多天内，我主要的精力和期待，都是在培养大家好的习惯、可以学会管理、学会主动学习等；虽然我会督促大家去准备英语等级考试，计算机等级考试，期末考试等，但我始终坚信，衡量大家价值和成功的地方不在于考试的成绩, 而在于是否发现了自己的潜力和兴趣，并积极主动的为之坚持与奋斗；接下来的时间我也会朝着这个目标不断去努力，为大家创造各种机会与条件，也希望能够看到大家都不断进步；人生的旅途中每一天都算数，希望大家珍惜好当下的每一刻，健康、快乐地奔向自己更加美好的未来，不只是大学四年，也希望在接下来的每一个阶段，都能够与大家分享彼此的体验，分享彼此的酸甜苦辣。
