一、关于新学期学习状态的调整
1.  认识本科阶段的重要性：本科阶段是个人成长的关键时期，很大程度上决定了未来的发展方向。虽然学习过程充满挑战，但克服困难后才能实现真正的进步。
2.  认真对待课堂教学：
    ◦   基本要求：新学期伊始，首先要配合好各门课程的教学，上课时认真跟随老师的节奏。
    ◦   调整心态：尽管认为传统课堂教学形式可能存在局限，但在无法改变的情况下，应积极看待。要相信老师们都认真准备了内容，听课过程本身是拓宽视野、发现兴趣点的重要途径。
    ◦   重新定义：将认真听课、跟上进度本身，也视作一种学习和“休息”的过程。
3.  高效利用课余时间：
    ◦   时间宝贵：本科是能依个人兴趣和意愿深入学习的关键阶段，此后的时间将更难集中。
    ◦   持续学习：无论是备战考研（如BI2023）还是其他年级，都有大量需要学习的内容。要利用好每一天，根据个人规划不断学习、反思和挑战自我。

二、关于SIG项目的安排与期望
1.  项目启动时间：暂未正式下发通知，预计需要一周到十天左右的时间进行准备。还需要做些准备，并针对上学期进展调整SIG课程。
2.  前期准备建议：在正式通知前，建议大家平时多积累、多拓展。明确自己感兴趣的方向，补充相关知识点。面对AI4Science等领域的快速发展，需要马不停蹄地学习。
3.  强调基础与逻辑：无论外界如何变化，都要打好扎实的基础（包括专业基础和逻辑思维能力）。没有好的基础，即使AI再强大，个人也难以形成主见并做好事情。
4.  对项目的期望与态度：
    ◦   本学期的重要性：本学期对师生都非常重要，从BI2022到BI025的同学都会参与SIG项目，期望能有所突破，产出一些实质性的好成果。
    ◦   摒弃应付心态：切勿将项目视为应付性任务，否则毫无意义。参与和推动项目的过程本身就是宝贵的学习和积累。

三、关于实验室使用的通知与要求
1.  教室安排：由于SKY@QLNU教室紧张，二号教学楼B座一楼的生物信息学实验室本学期被安排了课程。
2.  替代方案：实验室有课时，同学们可选择去图书馆、其他空闲教室或宿舍学习。
3.  安全与卫生要求：
    ◦   人人有责：在实验室时，必须注意安全和卫生。各年级班委要有计划地组织安排好。
    ◦   基本素养：维持好公共环境是对每个人最基本的要求，这关乎基本的为人素养。
四、新学期期待
    ◦   明确目标：每个人都需要进一步梳理自己的方向和目标，一个明确、严谨且能动态调整的目标至关重要，没有具体目标很难取得进步。
    ◦   注重平时的阅读与积累，不要让自己受限于狭窄的空间。    
    希望通过本学期，每位同学都能在SIG等项目中获得真实收获，实现个人突破和显著进步。
