# Exchange
为方便与BIers进行交流，设立了Exchange项目；如果学生有相关问题需要与指导教师进行沟通，或者需要指导教师进行远程指导等，可以与指导教师预约时间进行Exchange；交流以线上为主。


## 如何预约

学生通过[Doodle](https://doodle.com/)的[群组投票](https://doodle.com/meeting/organize/groups)，选择自己适合的时间区段（可以多选个时间段，根据交流的大概时间长段，可以在一天或几天内多选几个时间区间），并将Doodle链接发送至 bi.qlnu@hotmail.com；指导教师会根据可选择的时间确定Exchange时间、之后把Exchange线上交流链接发送给相应同学。


## 说明
- Exchange时间不限，以简洁、高效为主，交流主题需要明确。
- 简明扼要的问题鼓励通过[My-Question](..\..\修业\Bioinformatics\专业学习导论\课程项目\My-Question\Readme.md)交流。

## 历史版本

### 版本-1 说明

>每位同学每个月与指导教师主动联系沟通（至少）一次（以线上为主），具体时间因人而异、确定时间后指导教师会发送日历链接给同学；具体交流以简洁为主，遵循3+2+x模式，各位在3分钟内先汇总一下最近的进展和问题（建议提前准备一份提纲或PPT），然后利用2分钟左右时间结合具体问题进行交流，如有需要、延长X分钟；希望同学每天都在成长，在正确的时间、做正确的事；无论指导教师（们）在哪（只要在地球上、活着），Exchange系列至少持续到同学毕业，毕业后是否延续就凭感情了。

本版本Exchange时间和形式过于死板，无论同学的主动性好或者差，都存在诸多问题，故进行了升级。
