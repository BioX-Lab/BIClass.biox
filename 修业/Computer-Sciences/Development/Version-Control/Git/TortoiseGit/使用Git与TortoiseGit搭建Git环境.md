# 使用Git与TortoiseGit搭建Git环境

> TortoiseGit只能在Windows环境下运行，MacOS等用户可以使用[SourceTree](https://sourcetreeapp.com/)等工具。



## Git下载与安装

从[Git官网](https://git-scm.com/)根据个人操作系统选择相应版本下载，一般选择最新版本即可。安装过程根据提示操作就行，一般使用默认设置即可。对于安装位置尽量选择一个全英文、并且没有特殊字符的路径。



## TortoiseGit下载与安装

从[TortoiseGit官网](https://tortoisegit.org/)选择相应版本下载。安装过程根据提示操作就行，一般使用默认设置即可。如果不熟悉英文，也可以下载相应的语言包。TortoiseGit安装完成后，相关git功能以通过鼠标右键菜单访问、使用。



## 项目克隆与开发

复制项目git地址后（如 https://gitee.com/biox-lab/biclass.biox.git 或者个人的其它项目等，可以使用https形式），在相应目录，通过鼠标右键菜单选择克隆，即可将服务器上的项目在本地克隆一份，之后就可以不断推动项目进展。
