# 在Gitee中Fork项目后如何更改项目名称和路径

在 Gitee 中 Fork 项目后，拥有的是原项目的一个独立副本，可以自由地修改这个副本的项目信息。

例如在Fork [My-Log-template](https://gitee.com/biox-lab/my-log-template)项目后，可以将仓库名称和路径中的template字样去掉，可以根据如下方式进行项目设置进行修改。

## 进入项目设置
- 打开Fork 项目页面，点击右上角的「管理」按钮
- 或直接访问：`https://gitee.com/你的用户名/my-log-template/settings`
