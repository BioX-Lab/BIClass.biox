# 如何将自学到的编程知识和理论有效转化为实际编码能力？

[BI2023/HCL/20240622/BioX]

“在大一自学完了Python和C语言的基础同时也学习了相关的数据结构与算法分析，但是没有办法将思路很好的用代码实现，如何将自学到的编程知识和理论有效转化为实际编码能力？”

对于程序设计，系统学习某种编程语言（如Python等），以及数据结构与算法等基础是必要的，但程序设计除了理论上的理解和认识外，也需要大量的实践练习，需要一个量变到质变的过程。
1. 需要先看几遍参考书，但不要贪多、先专注到一门编程语言即可（首推Python来入门），当熟悉了一门编程语言之后，很容易迁移到其它语言；实际上除了项目或者工作需要，把一门语言弄通即可，后续随着领域的发展，再有计划学习新的语言。
2. 在阅读过程中，可以把参考资料中的示例在本地电脑上练习、实践一下，不只是把代码原封不动复制到开发环境中，还要通过调试，理解每一句代码的实际意义。
3. 在学习的同时，再结合实际项目进行更多实践，遇到问题时再回到相应知识点，认真学习；在BI@QLNU，可以在生物信息学进（高）阶与实践课程（即SIG项目）中多实践，学会用程序去解决问题；一般来说，一个人在实际项目中应用到的代码类型是有限的，甚至就是固定的几种，只要一个程序运行通了，后续就更加容易提升。

坚持下去，让编程变成一个人很重要的工具和伙伴，但这个顿悟（有“感觉”）的过程需要时间，不要着急；行则将至。
