#  如何选择Markdown编辑器

Markdown作为一种标记语言，是程序员或创作者的首选，也是AI时代一种优质的沟通方式。作为文本格式，可以使用Notepad (++) 等编辑器，但最好使用相对专业的编辑器，目前市面上相关Markdown编辑器数量“成百上千”，这里分几个层次简要介绍如何选择相关的Markdown编辑器，结合个人喜好和经验进行选择（欢迎诸位分享相关经验: bi.qlnu@hotmail.com）。

## Markdown文档

如需快速、便捷查看或者编辑某一或几个Markdown文档时，优先选择在本地电脑安装或者线上使用如下某种编辑器。

[marktext/marktext: 📝A simple and elegant markdown editor, available for Linux, macOS and Windows.](https://github.com/marktext/marktext)

[Typora — simple yet powerful Markdown reader.](https://typora.io/)
笔者最喜欢的一款编辑器，缺点是已经开始收费，虽然价格不高，仍然可以使用V1.0之前的免费版（可以从网络或者[BIClass项目](https://gitee.com/biox-lab/biclass.biox/)配套网盘中获取）。

### 线上编辑器

如下工具支持线上编辑。

[StackEdit – In-browser Markdown editor](https://stackedit.io/)

[Cmd Markdown 编辑阅读器 - 作业部落出品](https://www.zybuluo.com/mdeditor)


## 知识型项目

当需要处理某一知识型项目时，可以使用如下编辑器，如BI-Note笔记项目、个人知识库等；大部分工具也支持线上使用。

[Obsidian - Sharpen your thinking](https://obsidian.md/)

[Notable - The Markdown-based note-taking app that doesn't suck](https://notable.app/)

[TriliumNext/Trilium: Build your personal knowledge base with Trilium Notes](https://github.com/TriliumNext/Trilium)

[Your One-Stop Publication Workbench | Zettlr](https://www.zettlr.com/)


## 代码型项目
如果项目涉及代码等，可以直接使用IDE进行Markdown编辑，在写程序的同时、便捷编辑相应Markdown文本。几乎所有IDE（或相应插件）都会支持Markdown，如下为几款常用的“编辑器”。

[Visual Studio Code - The open source AI code editor](https://code.visualstudio.com/)

[TRAE - Collaborate with Intelligence](https://www.trae.ai/)

[Cursor：用 AI 编码的最佳方式](https://cursor.com/cn)

[Project Jupyter | Home](https://jupyter.org/)
