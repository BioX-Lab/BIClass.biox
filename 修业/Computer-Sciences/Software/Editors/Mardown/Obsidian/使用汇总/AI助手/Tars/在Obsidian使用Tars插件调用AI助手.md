# 在Obsidian使用Tars插件调用AI助手

## 安装插件
在Obsidian中，点击“设置”图标，选择“社区插件”，点击“浏览社区插件”，搜索“Tars”插件并安装。
## 获取API密钥
根据你想要使用的AI服务，访问其官方网站或API平台，注册账户并登录，在API管理页面创建一个新的API密钥。
## 配置插件
回到Obsidian，进入“社区插件”设置，找到并选择Tars插件，在插件设置中，点击“添加新的AI助手”，从下拉菜单中选择你想要使用的AI服务，在弹出的配置窗口中输入你的API密钥。
## 使用
打开或创建一个新的Obsidian笔记，输入触发AI的标签，如“#DeepSeek”，然后输入你想要AI助手回答或生成的问题或提示，之后，Tars插件会根据你的输入生成文本。

### 示例

``` text
#我 : 一段话介绍一下生物信息学（用户消息）
(隔开一个空行) 
#DeepSeek : （触发）
```

#Obsidian #Tars #AI助手
