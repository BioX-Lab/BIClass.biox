# 如何清除 Word 文档中的详细信息属性

可（推荐）通过文件属性对话框实现：
1. 找到要清除属性的 Word 文档，右键点击该文档。
2. 选择“属性”。
3. 在弹出的属性对话框中，切换到“详细信息”选项卡。
4. 点击下方的“删除属性和个人信息”链接。
5. 在打开的“删除属性”对话框中，可以选择“创建不包含任何可删除属性的副本”，这样会生成一个新的文档，原文档保持不变且新文档不包含可删除的属性；也可以选择“从此文件中删除以下属性”，然后勾选要删除的属性，如“Details”相关内容等，点击“确定”即可。
也可以借助第三方工具实现。
需要注意的是，在清除属性前，请确保你不会误删重要的信息，并且最好对文档进行备份，以防万一。
