#  使用Nano编辑器保存并退出文件
## 保存文件
在 nano 编辑器界面中，按下`Ctrl + O`（这里的 O 是字母 O，代表 Output，也就是输出保存的意思）组合键，编辑器底部会提示你输入要保存的文件名，如果你编辑的是已有文件，那么文件名通常已经自动显示在那里了，直接按下 Enter 键就可以保存文件内容到原文件中。如果是新建的文件，你需要输入想要保存的文件名（若需要保存在特定目录下、也输入完整路径），然后按`Enter`键完成保存操作。
## 退出编辑器
在成功保存文件后，按下 `Ctrl + X`（X 代表 Exit，即退出的意思）组合键，这样就可以退出 nano 编辑器，返回到系统命令行界面了。
