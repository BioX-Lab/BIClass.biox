# 使用夸克作为PDF轻量阅读器

夸克深度融合千问AI后，可选择作为轻量级PDF阅读器，无需额外工具/插件，原生支持PDF的打开、搜索、阅读及批注、编辑、格式转换等操作；搭配千问AI可实现智能辅助处理，界面极简无广告、数据安全合规，能高效满足PDF阅读与知识管理需求。
## 夸克安装
- 手机端：应用商店搜索「夸克」，点击安装，完成后直接打开即可。
- 电脑端：浏览器搜索「夸克官方下载」或进入[官网](https://www.quark.cn/)，选择对应系统版本下载安装包，双击后按提示完成安装。
## 使用方法
- 直接打开：本地或网络获取的PDF文件，无需插件可直接打开，快速访问阅读。
- 搜索阅读：通过夸克搜索功能查找互联网PDF文件，点击搜索结果链接即可打开阅读。
- 批注与编辑：阅读界面点击「批注」可添加阅读笔记；点击「编辑」能对PDF内容进行基础标注修改。
- 格式转换：通过「转格式」功能，可一键将当前PDF转为Word、图片等格式，操作在阅读界面直接完成。
- 即时翻译：点击「翻译」功能，可对PDF中的单句或整段文字进行实时翻译。
- AI辅助阅读：阅读时划词/选段唤起千问AI，可实现术语解释、段落翻译、内容总结等操作。
