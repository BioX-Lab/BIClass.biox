# AI

## 写在前面

AI时代已经来临，它正在重构我们的世界，包括BI。做为一个内心有温度的人，应该要积极拥抱AI，与AI为伴。AI是一个极度聪明的“朋友”，但它的丰富与性格，需要我们和它像朋友一样，去交往、去了解它，同时让它去成就更好的我们。
