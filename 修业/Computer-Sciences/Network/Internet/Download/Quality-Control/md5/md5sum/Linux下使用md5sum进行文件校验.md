# Linux下使用md5sum进行文件校验

在Linux操作系统中，`md5sum`是一个非常实用的工具，用于生成和校验文件的MD5（Message Digest Algorithm 5）散列值。MD5是一种广泛使用的密码散列函数，可以产生一个128位（16字节）的散列值，常用来确保数据的完整性和一致性。

Linux中的md5sum命令，用于生成和校验文件的md5值，以确保文件在传输或存储过程中的完整性。
## 生成文件md5值
`md5sum file`，如`md5sum data.zip`。

## md5校验
使用`-c`选项，根据已生成的md5校验当前文件，一致返回`OK`，否则返回错误信息，如`md5sum -c d.md5`；修改文件后md5会变化，校验会失败，如`md5sum -c data.md5`；使用`--status`可通过命令返回值判断校验结果。

## 重要参数
- -b：以二进制模式读入文件内容。
- -t：以文本模式读入文件内容。
- -c：根据已生成的md5值对现存文件进行校验。
- --status：校验完成后不生成提示信息，通过命令返回值判断，校验一致返回0，不一致返回1。
## 说明
- md5sum校验文件内容，与文件名无关。
- 文件越大，md5sum校验时间越长。
- md5值是128位二进制数据转换的32位十六进制值，不过md5校验有极小概率不同文件生成相同值，`SHA*`系列算法更安全。
