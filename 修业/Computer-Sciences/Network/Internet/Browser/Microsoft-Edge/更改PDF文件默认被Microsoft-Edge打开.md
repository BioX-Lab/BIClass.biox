# 更改PDF文件默认被Microsoft Edge打开

Windows系统中如果出现PDF文件默认被Edge浏览器打开，可以通过如下方式更改设置。

## 方案 1

选中待打开的PDF文件，右击打开`属性`，在`General`选项卡中点击`Opens with`右侧的`Change...`按钮，在弹出的对话框中选择相应的PDF阅读器，选择`OK`；最后点击`属性`对话框下侧的`OK`按钮完成设置。

## 方案2
也可以通过Windows中的`设置`进行更改。
- 点击电脑左下方Windows图标，在弹出的窗口选择设置选项，齿轮形状的选项。
- 在设置窗口页面里，选择应用，点击进入应用窗口。
- 在应用窗口里，在左侧的选项里选择第二个选项，默认应用选项。
- 鼠标移至右侧窗体部分，滑动页面至下方，看到按应用设置默认值选项。
- 进入按应用设置默认值窗口后，鼠标下滑，找到Microsoft Edge应用，然后点击管理按钮。
- 在打开的Microsoft Edge管理页面里的左侧找到PDF选项，右侧对应的默认打开软件，点击该软件弹出一个窗体进行选择你想默认打开PDF文件的应用即可。
