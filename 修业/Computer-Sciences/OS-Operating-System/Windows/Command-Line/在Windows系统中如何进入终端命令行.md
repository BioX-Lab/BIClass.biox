# 在Windows系统中如何进入终端命令行

## 命令提示符 (CMD)
1. **快捷键**：`Win + R`，输入 `cmd`，按回车。
2. **开始菜单**：点击开始按钮，输入 "cmd" 或 "命令提示符"。
3. **文件资源管理器**：在地址栏输入 `cmd` 后按回车。

## Windows PowerShell
1. **快捷键**：`Win + X`，调出“快速访问菜单”，选择 "Windows PowerShell"（或通过右击开始按钮实现）。
2. **开始菜单**：点击开始按钮，输入 "PowerShell"。
3. **右键菜单**：在文件夹空白处按住 Shift 键右键，选择 "在此处打开 PowerShell 窗口"。

## Windows Terminal
- 从 Microsoft Store 安装 Windows Terminal；
- 快捷键：`Win + R`，输入 `wt`。
