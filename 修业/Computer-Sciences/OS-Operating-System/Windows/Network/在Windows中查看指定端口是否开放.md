# 在Windows中查看指定端口是否开放

可以通过如下方法查看Windows系统中指定的端口是否开放，以9943端口为例。

## 使用netstat命令查看
打开命令提示符（可以通过在开始菜单中搜索“命令提示符”并打开，或者按下 `Win + R` 组合键，输入“cmd”后回车来打开），在命令提示符窗口中输入以下命令：

``` bash
netstat -ano -p tcp | find "9943"
```

执行上述命令后，根据返回结果来判断端口是否开放：
- **如果未显示与 9943 端口相关的信息**：则表明当前系统中 9943 端口未处于开放状态。
- **如果显示了包含 9943 端口相关的详细信息（如对应的进程 ID 等）**：则意味着该端口处于开放状态。

## 使用telnet命令查看
`telnet` 命令同样可用于检测端口是否开放，不过需要先确保 `telnet` 客户端功能已启用。
### 启用 telnet 客户端功能
1. **打开运行窗口**：按下键盘上的 `Win` 键，同时按下 `R` 键。
2. **打开 Windows 功能对话框**：在弹出的运行对话框中输入 `optionalfeatures`，然后点击“确认”按钮，此时会弹出 Windows 功能对话框。
3. **勾选 telnet 客户端**：在 Windows 功能对话框中，仔细查找“telnet 客户端”这一选项，在其前方的空格处打上勾进行选择，之后点击“确认”按钮，系统就会自动安装并开启 `telnet` 客户端功能。

### 使用 telnet 命令检测端口
待 `telnet` 客户端功能开启后，再次打开命令提示符窗口，输入以下命令：

```bash
telnet 127.0.0.1 9943
```

输入上述命令后，如果端口是开放且连通的，会出现另外一个黑色的 `dos` 窗口；反之，如果端口未开放或者无法连通，则会提示连接失败等相关错误信息。
