# Math
> - 数学作为形式科学的重要方向，是科学技术的基础，也为人类提供了重要的思维方式和工具。
> - 数学的学习应该是鲜活的（不只是做题，这是一类低层次的数学实践），它涉及到人类生活的方方面面，当然也包括生命科学；数学与生命科学的相互影响会逐渐加深，数学也是生命科学正式进入理论科学的重要基础。
> - 数学的学习应该是终生的，它对智力的锻炼和认知水平的不断提升是不可或缺的。
>
