# My-Profile

> My-Profile包含档案等个人资料。
>
> 通过My-Profile, 每学期及时更新一下自己的相关信息、包括个人的一些相关成就（比如证书、比赛、奖励等，也及时更新在个人档案中），在模板文档的基础上，个人随时维护好排版等；My-Profile一方面是学生个人的系统总结，另一方面从专业培养的角度，也会基于My-Profile为学生的就业和学业深造等提供更加精准的指导与推荐等。



## 重要节点

### 新建项目

- 在个人[Gitee](https://gitee.com/)中新建"My-Profile"项目，项目访问权限设置为“私有“；
- 添加指导教师guangyou_duan/rabbit-feather（成员权限设置为“开发者”即可）。



## 项目评价

根据项目内容评价学生的结合表现等。
