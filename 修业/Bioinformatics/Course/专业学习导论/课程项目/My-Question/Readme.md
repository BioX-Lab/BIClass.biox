# My-Question



## 项目介绍

- 通过My-Question，每天提问至少一个（实际开展过程中，个数不限）与专业、个人发展或者其它任何方面相对合理的问题；
- 提问对个人是一个很好的梳理、同时也是一个宝贵的财富。每天不需要提问太多，只认真思考一个问题就足矣，当然问题的个数不限；坚持下去，会对自己有一个更深刻的认识（大概率也会清楚自己的无知、同时也会清楚自己不知道自己的无知）。
- 指导老师会有选择回答相应问题，部分问题会通过BioX生物信息学等平台发布。
- 在个人[Gitee](https://gitee.com/)中新建"My-Question"项目，项目访问权限设置为“私有“；
- 添加指导教师guangyou_duan/rabbit-feather（成员权限设置为“开发者”即可）。
- 项目内部，所有的问题可以放在一个文件中，也可以分门别类放在不同的文件中，按自己的逻辑归类整理即可。
