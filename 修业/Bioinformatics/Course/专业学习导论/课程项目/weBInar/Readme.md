# weBInar

> weBInar是一个分享平台，以锻炼BIer分享为主；也会不定期邀请外报报告人参与。
> **关于发音**：可以采用webinar（“美/ˈwebɪnɑːr/”）发音，也可以采用发音组合we（“美/wi; wiː/”）+BInar。



## 运行规范

- weBInar鼓励BIers毛遂自荐参与分享，确定了要分享的主题后可随时向 bi.qlnu@hotmail.com 发送申请邮件；申请通过、确定加入weBInar分享列表后，如有需要、指导教师会全程指导、辅助。
- weBInar会以专业相关为主，但**主题不限**，只要是喜欢的或者擅长的方向，无论理论、技术、硬件、软件或者兴趣等都可以在weBInar中一展风采；weBInar不求大而全，只要讲通一个问题或者技术就已经很好，建议有实践或互动环节；
- weBInar以实行线上直播方式为主（具体通知见BI_QLNU共享日历），如果需要线下会另行通知；采用线上形式时，如果报告人没有合适报告地点，可以选择L2B127@QLNU（其它自习等活动需要让位给weBInar采用）。
- 每次都会为weBInar的报告人发放“专家费”或者是小的鼓励，目前执行的标准是一节分享200元，后续评估后、再进行调整；经费目前由指导老师自筹；报告结束后，（仅第一次填写即可）请及时填写[个人银行账号等信息](https://gqfwpj47z2n.feishu.cn/share/base/form/shrcnL1vn3zyM21cLaIXohpcQnb)，仅用于后续补助发放（不另作它用），另外发送一份个人签名图片（背景透明）至 bi.qlnu@hotmail.com；
- 报告人在准备过程新建一个weBInar的Gitee项目（如weBInar-主题等），用于准备相关主题的报告，添加指导教师(guangyou_duan)为项目成员，便于指导教师及时指导；（可选项，但鼓励）
- 与指导教师确定相应报告可以安排上线后，通过[Doodle](https://doodle.com/)的[群组投票](https://doodle.com/meeting/organize/groups)，选择几个自己适合的时间区段发送给 bi.qlnu@hotmail.com 、确定最终的时间；
- 每次weBInar报告结束后，报告人与指导教师会将相关材料进行内部分享，以供BIers查阅、重复学习；
- （如果组织在线报告时）内部报告人 (BIers@QLNU)安排好人员进行录屏，报告结后将录屏链接发送给bi.qlnu@hotmail.com；
- 鼓励所有在校BIers时间合适时、尽量参与weBInar讲座，相互学习。



## 专业学习导论课程定制版

weBInar除了按规范正常推动外，在开设专业学习导论相应学期，进行一次weBInar分享的简化版。

### 具体要求

- BIer根据自己的兴趣和学习情况，制作视频、分享一个知识点，视频长度5分钟左右（可根据实际情况调整），相应视频会上传到“BioX生物信息学”**抖音**平台；
- 提交内容包括两个文件，一个是视频文件，一个是文案文件，文件命令统一为"学号-姓名.后缀名"（学号、姓名替换为自己的信息，后缀名为mp4、docx、md等视频或文档格式）；文案文件包括视频作品的描述和关键词（至少两个关键词），关键词会用于话题创建、作品检索等；无论是视频还是文案，请精心准备，“BioX生物信息学”平台会按照提交内容原样发布。
- 对于文案文件，包含标题（30字以内）、介绍（1000字以内，不建议太长）、标签 5个左右（按重要性排序）；
- 对于视频文件，视频大小尽量控制在1G以内；
- 除视频与文案文件外，尽量提供封面图片两张（横封面4:3、竖封面3:4），文件名与视频、文档相同。
- 打分包括指导教师和抖音评价两部分，分别占40%和60%；其中抖音评价主要采用点赞、收藏、转发、平均播放时长、播放量五部分，各占$\frac{1}{5}$；
- 视频提交截止时间为相应学期圣诞与元旦之间，具体提交形式另行通知。
