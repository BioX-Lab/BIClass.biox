# My-Log

> My-Log用于学生在不同时间尺度上对个人学习与实践的规划与总结等，例如每日、每周、每月、季度、年度等。



## 重要节点

### 新建项目

- 在个人[Gitee](https://gitee.com/)中新建"My-Log"项目（参考[模板项目](https://gitee.com/biox-lab/my-log-template)），项目访问权限设置为“私有“；
- 添加指导教师guangyou_duan（成员权限设置为“开发者”即可）。


## 项目评价

根据项目内容评价学生的学习规划与总结等。


## 示例项目
[GYD-Log: GYD-Log项目，用于个人规划与总结。](https://gitee.com/biox-lab/gyd-log)

## 说明

- 第一学期课程开展过程中，需添加指导教师作为项目成员；课程完全结束后（比如学期末、看到课程成绩后），可以不与指导教师分享（鼓励继续与指导教师交流）。
