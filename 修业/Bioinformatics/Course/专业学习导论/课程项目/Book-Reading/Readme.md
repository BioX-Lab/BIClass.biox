# Book-Reading

> Book-Reading用于书籍阅读和相关感悟分享；在QLNU生物信息学专业，每个班级会坚持每天分享一本书，在班级内部进行分享交流，平均每人一个月左右会分享一本自己阅读过的图书，**!!!可以把个人轮值的日期在Calendar中标注好!!!**，“无限”循环下去，直至本科毕业。



## 重要节点

- 指导教师会将同学拉入班级的Book-Reading项目。



## 项目评价

根据分享内容评价学生的阅读、思考等。



## 说明

- 分享文件命名方式为: 分享日期-书名\_出版年份\_出版社_分享者.docx (md)，如“20221005-收获之神-生物技术-财富和食物的未来\_2006\_上海科学技术出版社\_孟柏芝.docx”，“20221020-王立铭进化论讲义\_2022\_新星出版社\_林欣源.docx”等。
- 书名中不用添加"《”“》"等字符，如果书名有标点符号，可以统一替换为"-"；出版社中的标点符号、空格（主要是英文）统一替换为”-“。
- 项目按年份、月份设置各级子目录，如2024/2025、2024/October、2025/July等，子目录建立由同学自主负责。
- `Book-Reading`为班级共同维护的项目，指导教师会在Gitee版中将相应同学拉入班级。
- 分享日期请参考相应班级的[分享值日表](分享值日表)。
