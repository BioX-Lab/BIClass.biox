# 生物信息学基础与实践
生物信息学基础与实践（Elementary Bioinformatics and Practice, EBP）课程围绕生物信息学相关的基础知识、技能与实践开设，自2021年秋季学期于QLNU生物信息学专业开设。


## 课程安排
- 自学《生物信息学》参考书以及BIClass项目中的推荐资料；
- 在Gitee中自主推动BI-Note项目（见“课程项目/BI-Note”目录）；
- 不定期举行线上（下）讲座。

## 课程评价与考核

### 过程性评价（60%）

结合BI-Note项目推动过程的规范性、投入程度等进行评价。

### 表现性评价（40%）

表现性评价主要针对BI-Note项目的具体内容、实际意义等进行评价。

注：括号内数值为相应指标所占的比重。
