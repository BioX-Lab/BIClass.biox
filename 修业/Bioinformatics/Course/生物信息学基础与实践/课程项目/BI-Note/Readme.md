# BI-Note

> BI-Note是生物信息学笔记项目，用于在日常学习与实践过程中不断积累、优化BI相关的知识、代码、经验等。


## 重要节点

### 新建项目

- 在个人[Gitee](https://gitee.com/)中新建"BI-Note"项目（参考[模板项目](https://gitee.com/biox-lab/bi-note-template)），项目访问权限设置为“私有“；
- 添加指导教师guangyou_duan（成员权限设置为“开发者”即可）。


## 项目评价

根据项目内容评价学生的学习、实践与认知水平等。



## 说明

- 第一学期课程开展过程中，需添加指导教师作为项目成员；课程完全结束后（比如学期末、看到课程成绩后），可以不与指导教师分享（鼓励继续与指导教师交流）。
