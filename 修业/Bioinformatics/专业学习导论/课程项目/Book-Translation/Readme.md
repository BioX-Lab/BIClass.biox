# Book-Translation

> 会为每一年级选取两本当年出版的英文图书（具体见“[Book-Translation图书汇总.md](Book-Translation图书汇总.md)”），学生根据喜好等选择一本进行翻译，鼓励两本都进行学习、翻译，另一本可以等学课程结束后进行翻译；同时鼓励大家阅读、翻译为其他年级推荐的图书。



## 重要节点

### 新建项目

- 在个人[Gitee](https://gitee.com/)中新建Book-Translation项目，项目访问权限设置为“私有“；
- 添加指导教师guangyou_duan/rabbit-feather（成员权限设置为“开发者”即可）；
- 在项目中新建一个目录，命名为相应图书文件名，进行相应翻译。



## 项目评价

根据项目内容评价学生的翻译成果等。



## 历史版本

- 本项目在BI2021/BI2022期间，是全班分工、共同翻译一本英文图书，实际效果不是很理想，另外工作（学习）量也较少，自BI2024/BI2023开始进行调整。



## 说明

- 翻译过程是一个不断优化的过程，鼓励大家通过翻译认真学习，结合个人理解，不断优化内容。
- 翻译可以采用word，markdown（鼓励）等格式。
