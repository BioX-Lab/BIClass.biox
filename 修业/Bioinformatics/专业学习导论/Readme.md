# 专业学习导论

> 课程前身“专业导论”，自2024年秋季学期开始升级为“专业学习导论”，除了对专业进行介绍外，也包括学习、做事、做人等内容。
>
> 2024年秋季以前，一直鼓励生物信息学专业的本科生坚持参与Book-Reading、My-Log、weBInar等项目，虽然这些“额外”的项目很重要，但没有在培养方案中体现，多少有些遗憾，借此课程升级之际，选取了六个项目纳入本课程，辅助学生更好地学习、成长，具体项目介绍和说明见“课程项目”目录。





## 课程安排

- 专业篇（第1周）

- 学习篇（第2周）

- 做事篇（第3周）

- 做人篇（第4周）

  >
  > - My-Question（第7周）
  >
  > - Book-Reading（第8-9周）
  >
  > - Book-Translation（第10-14周）
  >
  > - My-Profile（第15周）
  >
  > - weBInar（第16周）
  >
  > 注:
  > - 具体项目细节见“[课程项目](课程项目)”，如无特殊标明，课程评价与考核会参考相应学期BIer对应项目的具体进展。
  > - 课程项目由BIer自主推动，会结合BIer的进展等进行1v1指导，有需要要时会提前通知、组织集体讲解。
