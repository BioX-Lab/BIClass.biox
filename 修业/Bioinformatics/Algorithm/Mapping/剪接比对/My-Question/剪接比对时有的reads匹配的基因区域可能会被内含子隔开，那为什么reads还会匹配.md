# 剪接比对时有的reads匹配的基因区域可能会被内含子隔开，那为什么reads还会匹配

[BI2021/DS/2021111/BioX]

真核生物的基因组结构复杂，外显子和内含子交替排列。在转录和剪接过程中，不同的剪接方式可以产生多种mRNA剪接异构体，增加了蛋白质的多样性。
剪接比对是指将测序 reads 与参考基因组进行比对时，能够跨越内含子、区域，只匹配外显子区域的过程，剪接比对适用于匹配到参考基因组的情况。
剪接比对软件，如STAR、TopHat2等，能够识别并处理剪接事件。这些软件允许 reads 在比对过程中跳过内含子区域，只匹配外显子序列；软件通过已知的剪接位点或者通过预测剪接位点来实现这一点。
