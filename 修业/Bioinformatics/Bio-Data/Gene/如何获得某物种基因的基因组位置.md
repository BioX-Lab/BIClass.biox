# 如何获得某物种基因的基因组位置

## 写在前面
对于感兴趣的基因，需要知道基因的名称或相应数据库的ID，如Entrez Gene ID或Ensembl Gene ID等。
以下使用人类基因EXOC7为例，Ensembl ID为ENSG00000182473。

## 在线数据库查询
优先使用基因组注释数据库的网页进行在线查询，如Ensembl、NCBI的Gene数据库、UCSC Genome Browser等。

### Ensembl数据库
1. 访问Ensembl网站(http://www.ensembl.org/)。
2. 在搜索框中输入基因名称或ID，并选择“Homo sapiens”作为物种。
3. 点击“Search”。
4. 在搜索结果中，点击基因名称。
5. 在基因的概览页面，找到“Location”部分，这里会显示基因在染色体上的位置。
### NCBI Gene数据库
1. 访问NCBI Gene数据库（https://www.ncbi.nlm.nih.gov/gene）。
2. 在搜索框中输入基因名称或ID，然后点击“Search”。
3. 在搜索结果中，点击您感兴趣的基因。
4. 在基因的概览页面，找到“Genomic context”部分，这里会显示不同基因组版本中基因在染色体上的位置，包括起始和终止坐标。
### UCSC Genome Browser
1. 访问UCSC Genome Browser(https://genome.ucsc.edu/)。
2. 在搜索框中输入基因名称或ID。
3. 点击“Search”进行搜索。
4. 在搜索结果中，可以选择适当的基因组版本，找到基因在染色体上的位置。

## 使用命令行或程序查询
如果对命令行或程序操作熟悉，特别是需要批量查询时，可以使用命令行或程序查询。


### 使用BioMart的R包
```R
library(biomaRt)
ensembl <- useMart("ensembl", dataset = "hsapiens_gene_ensembl")
genes <- getBM(attributes = c("ensembl_gene_id", "chromosome_name", "start_position", "end_position"),
               filters = "ensembl_gene_id",
               values = "ENSG00000182473",
               mart = ensembl)
print(genes)
```
在这个例子中，`ENSG00000182473`是基因的Ensembl ID。您需要替换为您感兴趣的基因的ID。
