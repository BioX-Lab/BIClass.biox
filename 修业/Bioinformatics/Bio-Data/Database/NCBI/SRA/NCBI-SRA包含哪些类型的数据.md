# NCBI SRA包含哪些类型的数据


SRA (Sequence Read Archive, https://www.ncbi.nlm.nih.gov/sra) 是一个由NCBI维护的大型公共高通量测序数据存储库。它接受来自生命科学各领域以及宏基因组和环境调查的数据。SRA 存储原始测序数据和比对信息，以增强数据的可重复性，并促进通过数据分析进行新发现。

NCBI SRA包含不同类型（层面）的数据，大体结构层次关系为：Studies（研究课题）-> Experiments（实验设计）-> Samples（样品信息）-> Runs（测序结果集）。
## BioProject
该类型前缀为PRJNA或PRJEB等；这是对整个生物研究项目的一个标识符。它提供了研究项目的总体概述，包括研究目的、实验设计的范围等信息。
### PRJEB
PRJEB通常是欧洲生物信息研究所（EBI）的欧洲核苷酸档案库（ENA）中生物项目的编号前缀。ENA收录了全球范围内科研人员提交的各种生物分子序列数据等。
### PRJNA
PRJNA是美国国立生物技术信息中心（NCBI）的生物项目数据库中的项目编号开头。NCBI的生物项目数据库涵盖了大量生物研究项目的信息，包括基因测序、功能研究等多方面的数据。

## BioSample
该类型前缀为SAMN等；用于标识和描述生物样本。这些样本是实验的基础材料，如组织样本、细胞样本或者个体生物等。
## Experiment
该类型前缀为SRX；代表一个实验，一次实验可以包含一个或多个样本进行一个或多个测序运行。SRX编号所对应的实验记录包含了实验设计的细节，包括所使用的测序技术（如Illumina测序、PacBio测序等）、实验条件（温度、试剂等）、样本的处理步骤（DNA提取方法、RNA反转录条件等）。
## Run
该类型前缀为SRR；对应具体的测序数据结果集，也就是实际的测序数据文件。它包含了测序产生的原始reads信息，如序列长度、质量得分等细节。一个SRR通常代表一个测序运行（可以是单端测序或双端测序）的结果。
## Study
该类型前缀为SRP；一个Study类似于BioProject，但更侧重于从测序研究的角度进行描述。它涵盖了研究的主题、目标以及实验设计的总体情况。SRP编号下包含了一组相关的实验（通过SRX编号关联）和测序数据（通过SRR编号关联）。
## Analysis
该类型前缀为DRX或DRR；用于标识对测序数据进行的分析工作。DRX对应分析实验，DRR对应分析运行。这些编号所关联的记录包含了分析的方法（如使用的软件工具、算法）、分析的参数（比对参数、变异检测阈值等）以及分析的结果摘要等信息。
