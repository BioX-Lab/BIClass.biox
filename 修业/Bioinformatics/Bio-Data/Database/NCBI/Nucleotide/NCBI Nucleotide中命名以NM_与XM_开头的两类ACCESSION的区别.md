# NCBI Nucleotide中命名以NM_与XM_开头的两类ACCESSION的区别

[FQS2021/JS/20241114]
在NCBI Nucleotide数据库中，”NM_“和”XM_“是两种不同类型的核酸序列编号，其中"NM_"代表的mRNA序列质量控制更严格；如下为两者区别的简要汇总。

## 序列来源及性质
- NM_（mRNA reference sequences）
这些序列代表经过实验验证的成熟mRNA转录本序列。它们通常是从已被充分研究的基因转录产物中确定的，准确性和可靠性相对较高。这种序列经过了严格的审核和验证流程，包括实验测定（如RNA测序实验）和对已有文献数据的核对。
- XM_（model mRNA reference sequences）
主要是计算机预测的mRNA转录本序列。这些预测是基于基因组序列信息和基因预测算法。当一个基因尚未被完全实验验证其转录情况时，计算机会根据基因的DNA序列特征（如启动子区域、外显子 - 内含子边界模式等）来推测可能的mRNA序列。由于是预测结果，XM_*序列可能与实际的转录情况存在一定差异，需要进一步的实验验证。

## 应用场景和可靠性程度
- NM_序列
 在基因功能研究和分子生物学实验中被广泛应用。如果研究人员想要了解一个已知基因的准确mRNA序列用于设计定量PCR引物、构建表达载体等实验，NM_序列是首选。因为这些序列已经在实验中被确认，使用它们可以提高实验的准确性和可重复性。例如，在研究某种疾病相关基因的表达差异时，使用NM_序列来设计检测mRNA水平的引物，可以更准确地反映基因的真实转录情况。
 由于其经过验证的性质，在注释基因数据库、构建基因表达谱以及比较不同物种间同源基因的mRNA序列等工作中，NM_序列也发挥着关键作用。
- XM_序列
更多地用于基因组注释的初步阶段或者在没有现成的NM_序列时作为参考。当一个新测序的基因组进行初步分析时，XM_序列可以帮助研究人员快速获得基因可能的转录本信息，从而为后续的实验验证提供线索。不过，在使用XM_序列进行研究时，需要谨慎考虑其预测的不确定性。例如，如果基于XM_序列设计实验，可能需要先通过其他方法（如RNA - seq验证转录本是否真实存在）来提高实验的成功率。

## 更新和维护机制
- NM_序列
随着新的实验证据的出现（如更精确的转录起始位点确定、新的可变剪接形式发现等），NM_序列会被及时更新。这个更新过程通常涉及对原始实验数据的重新评估和验证，以确保序列的准确性。更新后的NM_序列会重新发布，并且会有版本号等信息记录序列的变更历史。
- XM_序列
 当新的基因组数据或基因预测算法改进后，XM_序列可能会被重新预测。如果预测的结果与之前的XM_序列有较大差异，也会进行更新。不过，由于其预测性质，更新的频率可能相对不稳定，并且主要取决于基因组研究和计算生物学领域的新进展。
