# 谷氨酸在等电点时，其$\alpha$ 羧基与$\gamma$ 羧基的解离比例多别是多少？( $\alpha$ 羧基pK为2.2，$\gamma$ 羧基pK为4.2，氨基pK为9.67)

[BI2021/QQL/20220417/BioX]

首先需要知道谷氨酸的等电点pI，然后使用Henderson-Hasselbalch方程来计算每个羧基的解离比例。
对于谷氨酸这样的酸性氨基酸，其等电点pI是两个羧基pKa的平均值：
$pI = \frac{2.2 + 4.2}{2} = \frac{6.4}{2} = 3.2$
在等电点pH 3.2时，谷氨酸的α羧基和γ羧基都部分解离。使用Henderson-Hasselbalch方程来计算每个羧基的解离比例。
对于α羧基：
$\text{pH} = \text{pKa1} + \log\left(\frac{[\text{解离的α羧基}]}{[\text{未解离的α羧基}]}\right)$
$3.2 = 2.2 + \log\left(\frac{[\text{解离的α羧基}]}{[\text{未解离的α羧基}]}\right)$
$\log\left(\frac{[\text{解离的α羧基}]}{[\text{未解离的α羧基}]}\right) = 3.2 - 2.2 = 1$
$\frac{[\text{解离的α羧基}]}{[\text{未解离的α羧基}]} = 10^1 = 10$
对于γ羧基：
$\text{pH} = \text{pKa2} + \log\left(\frac{[\text{解离的γ羧基}]}{[\text{未解离的γ羧基}]}\right)$
$3.2 = 4.2 + \log\left(\frac{[\text{解离的γ羧基}]}{[\text{未解离的γ羧基}]}\right)$
$\log\left(\frac{[\text{解离的γ羧基}]}{[\text{未解离的γ羧基}]}\right) = 3.2 - 4.2 = -1$
$\frac{[\text{解离的γ羧基}]}{[\text{未解离的γ羧基}]} = 10^{-1} = 0.1$

在等电点pH 3.2时，α羧基的解离形式是未解离形式的10倍，而γ羧基的解离形式是未解离形式的0.1倍。
