# 基因芯片中探针与基因的关系
[BI2022/RY/20241123]

基因芯片中探针是一段人工合成的单链DNA或RNA片段，其序列是根据已知的基因序列设计的，长度通常在几十到几百个核苷酸不等。
由于基因的复杂性，基因芯片中基因与探针的关系有一对一、一对多、多对一等多种情形，具体对应关系在基因芯片设计时已确定（包含在相应芯片的注释文件中，但随着基因注释质量的改进，对应映射关系等会有变化和调整的可能）。
下例`demo.csv`为一个基因芯片表达数据示例，包含两个芯片`chip1`和`chip2`，每行代表一个探针，`ID`为探针编号，`Gene symbol`为对应基因名称。

当计算一个基因的表达量时，可以求所有相关探针的平均值等，以下为R代码实例。
``` R
# 读取数据
x0=read.csv("demo.csv",header=T,check.names=F)
# 计算gene_symbol在不同芯片中的表达量
gene_symbol="ACTB"
# gene_data为gene_symbol在两个芯片中的平均表达量
gene_data <- colMeans(x0[grepl(gene_symbol,x0$"Gene symbol"), c("chip1", "chip2")])
```
