# BIClass项目评价标准

## 项目选题评分标准

考核能否结合课程与专业核心要求，设计出合理可行、兼具创新性与针对性的项目题目，重点评估选题的可行性、创新性及与专业要求的契合度。

| 总分值  | 评价标准        | 具体分值  |
| ---- | ----------- | ----- |
| 100分 | 选题与专业要求的契合度 | 0-30分 |
|      | 选题的可行性      | 0-30分 |
|      | 选题的创新性与针对性  | 0-20分 |
|      | 选题价值的论证清晰度  | 0-20分 |

## 任务规划评分标准

考核针对选定的项目题目和预设目标，能否清晰梳理项目核心方向，科学规划出贴合目标、逻辑清晰、可落地执行的对应任务，确保任务规划与选题目标高度匹配。

| 总分值  | 评价标准             | 具体分值  |
| ---- | ---------------- | ----- |
| 100分 | 项目核心方向的梳理清晰度     | 0-25分 |
|      | 任务规划与选题目标的贴合度    | 0-30分 |
|      | 任务规划的逻辑严谨性、层次清晰度 | 0-25分 |
|      | 任务的可落地性          | 0-20分 |

## 任务执行评分标准

考核能否严格按照创建的任务规划，有逻辑、规范地推进任务落地，高效解决执行中的各类问题，确保任务完成的规范性、完整性与逻辑性。

| 总分值  | 评价标准                     | 具体分值  |
| ---- | ------------------------ | ----- |
| 100分 | 是否严格按照任务规划推进执行           | 0-20分 |
|      | 任务执行的规范性、逻辑清晰度           | 0-20分 |
|      | 执行中各类问题的解决能力（自主解决/指导下解决） | 0-20分 |
|      | 任务完成的完整性、质量高低            | 0-20分 |
|      | 是否严格按照任务规划推进执行           | 0-20分 |

## 项目呈现评分标准

考核能否对项目最终成果进行科学梳理、规范呈现，重点评估成果的完整性、逻辑连贯性、内容准确性及呈现的规范性，体现成果与选题、任务的一致性。

| 总分值            | 评价标准            | 具体分值  |
| -------------- | --------------- | ----- |
| 100分           | 成果梳理的科学性、呈现的规范性 | 0-20分 |
|       | 成果结构的完整性、逻辑连贯性 | 0-25分           |
|       | 成果内容的准确性、充实度   | 0-25分           |
|       | 成果与选题、任务要求的一致性 | 0-20分          |
|       | 核心成果的突出度、清晰度   | 0-10分           |
