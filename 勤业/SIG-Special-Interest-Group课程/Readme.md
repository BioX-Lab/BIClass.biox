## 关于SIG项目的说明

- SIG-0（对应生物信息学基础与实践2课程）在第2学期开展；选题可以针对生物信息学（实验）教材、已发表中文论文中已有的项目进行复现与改进，也可以结合个人实际情况，参考SIG-1~6进行选题。
- SIG-1~4（对应生物信息学进阶与实践1~4课程）分别在第3~6学期开展；选题时针对个人的基础和兴趣，可以从已发表论文、书籍中已有项目的复现与改进、结合自身专业兴趣自主设计选题等角度进行选题。

- SIG-5~6（对应生物信息学高阶与实践1、2）是生物信息学进阶与实践的拓展，在第7、8个学期开展，个人可以进一步结合自身对生命科学的兴趣方向或者依据后续职业规划和进修方向进行自主选题。

## SIG拓展系列

### SIG-x
> 即毕业论文（设计）项目。由”guangyou_duan“指导的毕业论文会基于SIG-x项目推动，确定毕业论文指导关系后、指导教师会通知新建该项目；**不适用于**其他指导教师指导的毕业论文。
> - 指导教师除了全程指导外，也会独立创建、同步开展SIG-x项目。

### SIG-s
>"SIG-s"为希望在BioX-Lab参与研发工作的专属SIG项目，特别是希望"guangyou_duan"指导或与其合作开展研发工作。
>- 指导教师一般只提供全程指导，个别项目（特别是指导教师的项目或主要由指导教师完成选题的项目）会参与到具体工作中。
>- 项目命名从SIG-s1开始，依次类推，SIG-s2/SIG-s3等。
>- 如果想开展SIG-s项目，类似每学期的SIG课程项目，自主创建SIG-s项目，并添加指导教师"guangyou_duan"为项目成员，指导教师会主动参与其中。

## 注意事项

- 个人git项目每天及时提交相应进展；commit message参考BIClass项目中”修业/Computer-Sciences/Development/Version-Control/Git/Git使用规范.md“中的推荐。
- 学期结束后，个人的SIG项目可以进一步完善。
- SIG课程无法补考，如果指导教师看不到对应学期的项目，只能进行重修；
- 复现参考书或者期刊论文中的内容虽然不鼓励、但完全没有问题，一定要清楚解决什么问题，要达到什么目标，即要有选题的完整构思，不只是去复现了一个不痛不痒的软件操作或者分析等操作（这些内容应该是平时坚持去学习、实践的）。
- 进行到SIG-5/6时，特别是SIG-5，会与考研等时间冲突，需要项目负责人更加合理地安排好时间，人生没有那么多合适。
