# BIClass.biox


##  写在前面

本项目（简称BIClass）为生物信息学（Bioinformatics，简称BI或者生信）学习分享与交流平台。

生物信息学是一个幸福的专业，也是一个成长、开放的专业；它是生命科学的重要组成部分，从数据、计算、理论等角度去探索与推动生命科学的发展。

本项目初衷用于齐鲁师范学院（简称师院）生物信息学专业人才培养，在生物信息学教学过程中也遇到了诸多困难或者挑战，诸如学习主动性（积极性）差等，为系统分享生物信息学相关知识，开设了BIClass项目；一方面期待师院的学生可以通过生物信息学专业有所收获与成长，同时更加期待通过BIClass与真正需要学习、使用或者对生物信息学感兴趣的朋友分享、交流。

作为一门交叉学科，真正入门生物信息学需要具备较好的自学与实践能力，同时需要良好的总结与成长意识。学习、做事、做人是人生的重要组成部分，BIClass力争通过这些角度与同仁一起围绕生物信息学进行探索与交流。

### 项目使用方法

- 推荐关注（“Watching”）本项目，并设置"关注所有动态“；在个人设置/消息中心/通知设置中开通邮件接收通知（通知设置中“站内信"与"邮件"通知只保留其一即可，建议选择邮件通知），后续项目相关更新可通过邮件及时查阅。
- 推荐在本地克隆本项目，便于在本地电脑系统查看相关内容。
- 本项目文档的排版主要采用markdown格式，建议使用[Typora](https://typoraio.cn/)等markdown编辑软件查看，有的文档会较长、推荐结合使用大纲视图模式。

### 参考资料命名
 本项目中的涉及的参考资料，特别是文献或者图书，为方便起见，主要采用如下命名方式，可结合本命名规则检索原始出处或者在BIClass.biox网盘中查阅等。

 - 英文期刊&会议论文: first-author-given-name\_year-of-publication\_journal-or-conference-full-name
 - 中文期刊&会议论文: 论文题目\_发表年份\_期刊或会议名称
 - 非正式报告或论文: 论文题目\_发表年份\_作者全名（-隶属单位）
 - 图书: 题目（-版本）\_出版年份\_出版社
 - 演讲报告: 报告题目\_演讲日期\_演讲者（-隶属单位）

 其它未提及参考资料的引用逻辑类似，另外为了便于检索必要时会在上述名称后添加标签信息，如"\_tag1_tag2_tagN"等。

## 历史事件

- 2021年9月，齐鲁师范学院开始招收第一届本科生，每年计划招生规模40人。
- 2022年11月，启动了“Bio-X_Learning-Center”项目（本项目前身），该项目已经完成了进化，变成了另一个”物种“。
- 2024年9月，BIClass（本项目）启动，继续围绕生物信息学，从学习、做事、做人等角度分享与交流。
