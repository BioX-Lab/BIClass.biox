# BIClass.biox


##  写在前面

本项目（简称BIClass）为生物信息学（Bioinformatics，简称BI或者生信）学习分享与交流平台。

生物信息学是一个幸福的专业，也是一个成长、开放的专业；它是生命科学的重要组成部分，从数据、计算、理论等角度去探索与推动生命科学的发展。

本项目初衷用于齐鲁师范学院（简称QLNU或者师院）生物信息学专业人才培养，在生物信息学教学过程中也遇到了诸多困难或者挑战，诸如学习主动性（积极性）差等，为系统分享生物信息学相关知识，开设了BIClass项目；一方面期待师院的学生可以通过生物信息学专业有所收获与成长，同时更加期待通过BIClass与真正需要学习、使用或者对生物信息学感兴趣的朋友分享、交流。

作为一门交叉学科，真正入门生物信息学需要具备较好的自学与实践能力，同时需要良好的总结与成长意识。学习、做事、做人是人生的重要组成部分，BIClass力争通过这些角度与同仁一起围绕生物信息学进行探索与交流。

## 历史事件

- 2026年4月，BIClass在Gitee/GitHub中同步更新，此外会在配套网盘中有同步更新。
- 2025年9月，齐鲁师范学院2025级生物信息学本科专业学生入校（招生规模30人），授予学位变更为”工学学士“。
- 2024年9月，BIClass（本项目）启动，继续围绕生物信息学，从学习、做事、做人等角度分享与交流。
- 2022年11月，启动了“Bio-X_Learning-Center”项目（本项目前身），该项目已经完成了进化，变成了另一个”物种“。
- 2021年9月，齐鲁师范学院开始招收第一届生物信息学专业本科生，每年计划招生规模40人。

## 写在后面

- 在齐鲁师范学院开设生物信息学专业以来，大概是我（“BIClass发起者”）人生中成长最多的阶段之一，收获或感悟很多。对待BIers@QLNU，应该算是在全力以“付”，想把每天的学习、专业相关的经历与感受尽可能多分享给BIers@QLNU，但现实有些让人失望，也让人多次“受伤”，这大概也是教育圈的现状，无论是教学的基本目的和形式、学习的基本状态、还是做人的基本原则，都有太多提升的空间（后续争取在[Bio-Sharing](为人\Bio-Sharing)等地方进行更多分享）；作为一名（算是）追求“完美”的人，希望从BI@QLNU出去的学生，有很好的未来和提升，希望BIers@QLNU都是健康、善良和成长的，虽然BIers@QLNU的表现差强人意，但这份冀望不会变化。虽然QLNU的条件有限（条件或环境的不理想对于一个坚定、成长的人不会有任何影响），我一直坚持思考、创造条件让BIers@QLNU可以不断成长，特别是从学习、做事和做人三个角度去分享。
- 教育的目的是去点亮他人，而不是强加于人（特别是高等教育阶段），不能因为“不完美”而放弃，或者花费太多心神在感受上，这些都是无意义的内耗；我一直坚信BI的未来和幸福，无论我未来是否继续维护BI@QLNU，但学习、做事与做人的“追求”是不会变的，也会坚持把过往BI相关的经验和新的体会在BIClass持续分享；发起BIClass更多是一份对自我的要求，把过往零散的分享进行系统总结和拓展，也是对现有和未来BIers@QLNU的一份承诺；如果能有一些点、可以影响到某些人，也是人生的一件快意之事，不必事事、人人面面俱到。期待在BIClass，与认可、学习或者使用BI的你们，一起“学习”，共同“做事”，努力“做人”。
